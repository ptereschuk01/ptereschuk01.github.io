1

git remote add origin <url_github>

 

Подключение удаленного репозитория (название origin), но оно может быть и другим, и репозиториев может быть и несколько.

2

git push -u origin <branch_github >

Л -> У

При первой загрузке изменений мы должны связать ветки.

Из ветки, в которой мы находимся и запускаем эту команду, мы создаем связь между нашей локальной веткой и удаленной веткой branch_github.

Иначе говоря, при первой загрузке изменений из локальной ветки main на удаленный сервер под названием origin мы хотим связать ветки main и branch_github (как правило тоже main)

3

git push

Л -> У

После установки связи межу лок. и удал. ветками используем просто команду git push

3

git pull

У -> Л

После установки связи … так же просто.
